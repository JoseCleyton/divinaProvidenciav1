Manual do Usuário

Login
 Instruções para realizar login

Entre no endereço da página inicial aqui ->
Na página de login que irá aparecer digite o seu login e sua senha e clique no botão login. Se os dados estiverem corretos o sistema irá redirecionar para o Painel de Controle.
Obs -> Uma vez feito Login não será necessário realizar esse procedimento novamente, pois o sistema guarda as credenciais de acesso.
Esse procedimento só será necessário se o Usuário realizar o processo de Logout.

Logout
 Instruções para realizar logout

No topo da página no sistema existe um ícone de Usuário.
Clique nele e um submenu irá ser mostrado com o ícone de Logout .
Clique nele o o sistema irá redirecionar para a página de Login.

Painel de Controle
 Instruções para o painel de controle

O Painel de Controle é composto por 4 cards e 2 gráficos

Pedidos Totais

Esse card mostra a quantidade de pedidos realizados até o momento

Caixa

Esse card mostra o valor total disponível no caixa

Pedidos Ativos

Esse card mostra a quantidade de pedidos que estão com status em aberto

Produtos

Esse card mostra a quantidade de produtos que estão cadastrados no estoque. Observação: ele não leva em consideração a quantidade de itens disponíveis

Gráficos

Os gráficos mostram os valores totais realizados em pedidos por mês
Menu
 Instruções para acessar o menu

Para acessar o menu é só clicar no ícone que está no canto superior esquerdo. Ao clicar o menu irá aparecer.
Pedido
 Instruções relacionadas a Pedido
Inserir Pedido
 Instruções para inserir um novo pedido
Obs: Para inserir um Pedido, é necessário que tenha, no mínimo, um Produto cadastrado, pois no campo selecionável o sistema exibirá os produtos cadastrados no estoque !!!

Clique no botão de menu. Clique na opção Pedidos e um submenu será mostrado. Clique na opção Inserir Pedido. O sistema irá exibir o formulário para inserir um novo pedido. Aguarde o carregamento dos produtos no campo de seleção. Após isso selecione um produto no campo selecionável, digite a quantidade desse produto e clique no botão Adicionar Produto. Faça isso para quantos produtos desejar. A quantidade dos produtos podem ser editadas pelo ícone de "lápis" na tabela dos itens adicionados. Após adicionar os produtos clique no botão Fechar Pedido". O sistema irá exibir um modal onde o campo Cliente deve ser preenchido, esse campo serve para identificação do cliente como por exemplo o nome. O campo Observações não é obrigatório porém serve para informações adicionais como cor, ou outras especificações. Após isso, clique no botão Realizar Pedido e o sistema mostrará o status da solicitação !
Listar Pedidos Abertos
 Instruções para listar pedidos em aberto

Clique no botão de menu. Clique na opção Pedidos e um submenu será mostrado. Clique na opção Listar Pedidos Abertos. O sistema irá exibir a lista de pedidos com o status aberto.

Listar Todos Pedidos
 Instruções para listar todos os pedidos

Clique no botão de menu. Clique na opção Pedidos e um submenu será mostrado. Clique na opção Listar Todos Pedidos. O sistema irá exibir a lista de todos os pedidos com status aberto, cancelado e finalizado.

Finalizar Pedido
 Instruções para finalizar um pedidio

Clique no botão de menu. Clique na opção Pedidos e um submenu será mostrado. Clique na opção Listar Pedidos Abertos. O sistema irá exibir a lista de todos os pedidos com status aberto. Na lista escolha o Pedido e clique no botão de cor verde com símbolo de check . O sistema exibirá um modal. Selecione a forma de pagamento:
Dinheiro > à vista ... Dinheiro > à prazo.
Na opção Dinheiro > à vista o pedido é finalizado e o caixa é movimentado.
Na opção Dinheiro > à prazo o pedido é finalizado e mas o caixa não é movimentado pois ainda não recebeu nenhuma parcela.
Caso a opção seja Dinheiro > à prazo escolha a quantidade de parcelas para habilitar o botão de Finalizar.
Clique em Finalizar.
O sistema irá mostrar o status da solicitação!

Visualizar Pedido
 Instruções para visualizar um pedido

Clique no botão de menu. Clique na opção Pedidos e um submenu será mostrado. Clique na opção Listar Pedidos Abertos ou Listar Todos Pedidos. O sistema irá exibir a lista dos pedidos.
Na lista escolha o Pedido e clique no botão de cor amarelo com símbolo de olho . O sistema exibirá um modal com as informações do Pedido.

Cancelar Pedido
 Instruções para cancelar um pedido

Clique no botão de menu. Clique na opção Pedidos e um submenu será mostrado. Clique na opção Listar Pedidos Abertos. O sistema irá exibir a lista dos pedidos com status aberto
Na lista escolha o Pedido e clique no botão de cor vermelho com símbolo de X . O sistema exibirá um modal. Clique no botão Cancelar.
O sistema mostrará o status da solicitação !

Produto
 Instruções relacionadas a Pedido
Cadastrar Produto
 Instruções para cadastrar um novo produto

Clique no botão de menu. Clique na opção Produtos e um submenu será mostrado. Clique na opção Inserir Produto. O sistema irá exibir o formulário para cadatrar um novo produto.
Digite o nome desse produto
Digite a quantidade que deseja desse produto
Digite o valor unitário -> Obs: DIGITE VÍRGULA OU PONTO APENAS SE EXISTIREM CENTAVOS NO VALOR. EXEMPLO: 10,50 ou1500,50
O sistema coloca ponto ( . ) automaticamente para separar os centavos e vírgula ( , ) para separar as casas de milhares !
Clique em Cadatrar Produto e o sistema exibirá um modal de confimação.
Clique no botão Confirma. O sistema mostrará status da solicitação !
Listar Todo Estoque
 Instruções para listar todo o estoque

Clique no botão de menu. Clique na opção Produtos e um submenu será mostrado. Clique na opção Listar Todo Estoque. O sistema irá exibir a lista de todos os produtos cadastrados.

Editar Produto
 Instruções para editar um produto

Clique no botão de menu. Clique na opção Produtos e um submenu será mostrado. Clique na opção Listar Todo Estoque. O sistema irá exibir a lista de todos os produtos cadastrados.
Na lista escolha o Produto e clique no botão de cor amarelo com símbolo de lápis . O sistema exibirá um modal com as informações do Produto.
Edite as informações necessárias e clique no botão Atualizar Produto.
O sistema mostrará o status da solicitação !

Caixa
 Instruções relacionadas ao caixa
Contas a Receber
 Instruções para contas a receber

Caso tenha alguma Conta a Receber o sistema irá exibir um card listando todos os clientes que estão em atraso. O card contém um botão de fechar. Ao atualizar a página esse mesmo card irá aparecer novamente. Esse card contém:

Nome do Cliente

Nome do Cliente preenchido na abertura do Pedido

Valor

Valor total a receber calculado na abertura do Pedido

Parcelas Pagas

Número total de parcelas que já foram pagas

Ação

Ação possível que o usuário poderá tomar. Clicando nesse botão Receber Valor o sistema irá exibir um modal onde o usuário poderá informar que receberá ou recebeu uma parcela, caso seja a última parcela o sistema irá dar baixa nas Contas a Receber desse pedido.
Movimentações
 Instruções para movimentações do caixa

Clique no botão de menu. Clique na opção Caixa e um submenu será mostrado. Clique na opção Movimentações e o sistema irá mostrar todas as movimentações realizada no Caixa.

Retirar Valor do Caixa
 Instruções para retirar valor do caixa

Clique no botão de menu. Clique na opção Caixa e um submenu será mostrado. Clique na opção Movimentações e o sistema irá mostrar todas as movimentações realizada no Caixa.
Na parte superior da página têm um formulário. Preencha com os dados necessário: Descrição da retirada e o Valor a ser retirado.
Clique no botão Retirar.
O sistema irá exibir um modal. Clique no botão Efetivar Retirada.
O sistema irá exibir o status da operação !
